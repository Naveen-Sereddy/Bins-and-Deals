Bins & Deals

A promotional deals and discounts platform designed to help users discover local savings and special offers.

Live Demo

🌐 https://binsanddeals.com

Features

* Responsive UI
* Modern React architecture
* Animated user experience using Framer Motion
* Mobile-friendly design
* Fast performance with Vite

Technologies

* React
* Vite
* Tailwind CSS
* Framer Motion
* Lucide Icons
